## 🚀 立即使用

### 安装
```bash
cd tools/generator
pnpm install
```

### 快速开始
```bash
# 初始化
lgen init

# 生成组件
lgen c -t vue -n MyButton

# 查看帮助
lgen --help
```

### 运行测试
```bash
pnpm test
```

### 构建
```bash
pnpm build
```
